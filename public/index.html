CS 5610 - Web Fundamentals - Spring 2016

The assignment complementing the coursework was a MEAN stack based Form Maker. This web app allows a user to create online forms which can be edited through the app. Form Maker lets you perform CRUD operations on the fields within each form through a mobile-friendly design.

My project for this coursework is a FitBit data visualization tool called PerformX. This web app lets a user sync their Fitbit and PerformX accounts to leverage real time physical activity data and plot graphs that gives you a glance of your fitness regimes over time.

Application Architecture

Both Form Maker and PerformX follows the MVC architecture pattern where the app is divided into models, views and controllers where models manipulate the data and contain the core app logic, views specify the display of application data and controllers act as a bridge between the views and controllers converting data into formats expected by these components.
